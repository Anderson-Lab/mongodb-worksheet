    Since this is our final worksheet, I wanted to do something a little different and special. You have the skills now that if your boss came to you and said investigate MongoDB, you would be able to install configure and test MongoDB. In other words, I want you to show off your mad skills.

    1. Write a set of instructions to install MongoDB on Ubuntu 14.04 (this is the OS you are running under vagrant).

    2. Once MongoDB is up and running, create a short tutorial that shows how you can insert and query the database. Please use the specific towns example from the textbook. Your queries don't need to be complex. Just show the basics like we did in class.

    Straight forward questions

    1. What type of database is MongoDB?

    2. What programming language does it use?

    3. How do you query MongoDB?

    4. What types of queries does it support and does it not support?

    5. Can it support nested data?

    6. What form is the data in the database?

    7. Compare and contrast MongoDB and Riak.
